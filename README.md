# nabinpoudel004.com

Personal portfolio website for Nabin Poudel.

**Live site:** https://nabinpoudel004.com.np (or https://navin99x.github.io)

## Structure

```
├── index.html      # Home page
├── about.html      # About page with contact form
├── blog.html       # Blog (coming soon)
├── projects.html   # Projects showcase
├── 404.html        # Custom 404 page
├── style.css       # Styles
├── robots.txt      # SEO
├── sitemap.xml     # SEO
└── images/
    └── photo.jpg   # Profile photo
```

## Tech Stack

- Vanilla HTML, CSS (no frameworks)
- Plausible Analytics (privacy-friendly)
- Formspree for contact form
